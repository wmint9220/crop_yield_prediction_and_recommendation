# 🌱 Crop Insight System

## 📌 Overview

Crop Insight is a machine learning–based web application that recommends suitable crops and predicts crop yield based on environmental and agricultural inputs. The system also provides trend visualization to support data-driven decision-making.


## ⚙️ Features

* Trend Visualization using dataset
* Crop Recommendation (Random Forest Classifier)
* Yield Prediction (XGBoost Regressor)
* Report Generation
* Interactive Web Interface (Streamlit)

---

## 🧠 Technologies Used

* Python
* Streamlit
* Scikit-learn
* XGBoost
* Pandas, NumPy, Matplotlib

## 🚀 How to Run the System

### Step 1: Clone Repository

```
git clone <your-github-link>
cd CropInsight
```

### Step 2: Install Dependencies

```
pip install -r requirements.txt
```

### Step 3: Run Application

```
streamlit run app.py
```

### Step 4: Open in Browser

The system will automatically open at:

```
http://localhost:8501
```
